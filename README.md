# Simple Tetris
Just made a tetris game for fun. Run with `./tetris`.

# Controls
- **UP**: Rotate shape.
- **LEFT**, **RIGHT**: Move the shape.
- **DOWN**: Accelerate the fall.
- **p**: Pause game.
- **ESC**: Exit game.
